My Projects

Library Management System

Built a Java-based Library Management System with MySQL database, featuring a simple GUI for managing books. The main problem the system solves is the manual management of library records, which can be time-consuming and prone to errors. By digitizing the process, the system improves record accuracy, simplifies book tracking, and enhances overall library management.

Live Demo GitHub

Exam Results Generator

Built a Spring Boot application that generates downloadable PDF exam result reports and has an emailing feature from a MySQL database using iText for dynamic formatting. HTML, CSS and JavaScript were used for the GUI. The application allows the students to download their results transcript or have it emailed to them on a dedicated platform

Live Demo GitHub

Smart Internship Placement System

The Smart Internship Placement System (SIPS) is a web-based platform designed to connect students with companies offering internship opportunities. The system is being developed using HTML, CSS, and JavaScript for the frontend, Spring Boot for the backend, and MySQL for the database.
